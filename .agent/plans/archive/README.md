# Archived Plans

> **Purpose:** Completed plans are moved here for historical reference.
> **Rule:** When a plan is 100% complete and verified, move from `.agent/plans/` to this folder.

---

## Archive Format
When archiving, add this header to the plan:
```markdown
> **Archived:** YYYY-MM-DD
> **Completed By:** @AgentName
> **Final Status:** ✅ VERIFIED
```
